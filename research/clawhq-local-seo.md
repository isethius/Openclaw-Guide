## 1. Google Business Profile — What Category?

**Primary category: `Computer support and services`**

This is the correct call. Here's why:
- "AI consultant" is not yet an established GBP category. Where it exists, it has low search matching volume.
- Google's algorithm maps "AI setup," "AI installation," "AI assistant for business" queries heavily to the **computer support** category cluster.
- `Computer support and services` is the broadest, most established match for on-site technical implementation work.

**Secondary categories to add:**
- `Computer service`
- `IT consultant` (if available in your GBP interface — search for it)
- `Technology company`
- If `AI services` appears in the GBP interface, add it as a secondary — but don't make it primary yet.

**GBP Profile Optimization Checklist:**
- [ ] Business name: "ClawHQ" (exact, no keyword stuffing — TOS violation)
- [ ] Primary address: Seth's Tucson address (service-area business)
- [ ] Set service area: Tucson, Phoenix, Scottsdale, Mesa, Tempe, Chandler, Gilbert
- [ ] Phone: one consistent number used EVERYWHERE
- [ ] Website: clawhq.com
- [ ] Hours: set realistic availability hours (even "by appointment")
- [ ] Description: Use "AI assistant installation," "OpenClaw setup," "Tucson," "Phoenix" naturally
- [ ] Services: Create individual services (AI Setup, OpenClaw Installation, Business AI Consultant, etc.)
- [ ] Photos: Upload real photos of the setup process, equipment, workspace
- [ ] Posts: weekly GBP posts about AI tips, local business benefits — keeps profile active
- [ ] Q&A: Seed your own questions ("Do you offer in-home AI setup in Tucson?") and answer them

**Google Maps 3-Pack Strategy:**
Google ranks local pack on 3 signals:
1. **Relevance** — keyword match (GBP description + website content + services)
2. **Distance** — proximity to searcher (Tucson address helps for Tucson searches)
3. **Prominence** — reviews, backlinks, website authority

The 3-pack is winnable in Tucson because nobody else is competing for these terms. Phoenix is competitive for general IT but not for AI installation specifically. Start in Tucson, dominate, use that authority to push into Phoenix.

> **Note on the "open" signal:** GBP increasingly surfaces businesses currently open. Set availability hours so you show as open during business hours.
